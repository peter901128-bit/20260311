# 이 폴더를 GitHub 저장소에 올리는 방법

1. Git 설치: https://git-scm.com/download/win

2. 터미널(Git Bash 또는 PowerShell)을 열고 이 폴더로 이동한 뒤 아래를 순서대로 실행하세요.

처음 한 번만 (저장소 연결):
  git init
  git branch -M main
  git remote add origin https://github.com/peter901128-bit/20260311.git

이후 업데이트할 때마다:
  git add -A
  git commit -m "변경 내용 설명"
  git push -u origin main

※ push 시 GitHub 로그인(또는 토큰)이 필요할 수 있습니다.
