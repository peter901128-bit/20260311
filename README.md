# 로또 번호 추천 서비스

베가스 슬롯머신 테마의 로또 번호 추천 웹 페이지입니다.

## 기능

- 1~45 중 무작위 6개 + 보너스 번호 추천
- 슬롯머신 레버로 번호 뽑기
- 슬롯 릴처럼 번호가 굴러가는 연출
- 한 번에 여러 세트 생성 (1~10세트)
- 최근 10회 1등 당첨번호 표시

## 실행 방법

`index.html` 파일을 브라우저에서 열거나, 로컬 서버로 실행하세요.

```bash
npx serve
```
